		\item \footnotesize \textbf{ExplodedView}: classe geral, responsável pelo por tratar os dados de entrada (tanto o modelo 3D quanto o \emph{input} do usuário).
		\item \footnotesize \textbf{Part}: classe que representa as partes do modelo.
		\item \footnotesize \textbf{SegmentedPart}: classe que herda de \textbf{Part} e representa os recipientes divididos. A divisão dos modelos é feita através de um \textbf{ClipNode} do \textbf{OSG}.
		\item \footnotesize \textbf{CollisionData}: representa os dados de colisão.
	\end{itemize}
